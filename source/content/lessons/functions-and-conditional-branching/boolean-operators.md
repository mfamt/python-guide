tldr: |
  
  ## `and` operator

  ~~~py
  True and True    #   True
  True and False   #   False
  False and False  #   False
  ~~~

  ## `or` operator

  ~~~py
  True or True     #   True
  True or False    #   True
  False or False   #   False
  ~~~

  ## `not` operator

  ~~~py
  not True               # False
  not (True and False)   # True
  ~~~
